# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | ✅ |

## Reporting a Vulnerability

**Do NOT open a public issue for security vulnerabilities.**

Please report security issues privately:

- **Email:** info@allsafe.com.ar
- **Response time:** 48 hours
- **Language:** Spanish or English accepted

Please include: affected version, steps to reproduce, and potential impact.

We take security seriously and will acknowledge your report promptly.
